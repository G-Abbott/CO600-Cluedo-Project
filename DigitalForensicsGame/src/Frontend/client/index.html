Digital Forensics Detective!

Digital Forensics detective is a spin off version of the original cluedo game. It uses characters and places from Kent University and adds a digital forensics twist. This includes various attacks such as DDoS and Phishing in place of the weapons that cluedo originally had.

About
A Gamemaster sets the scenario and the users load in to the game server side using any web browser. The scenario includes the attacker who did the crime, the type of attack they used and the room they did it in. The players roll the dice to acquire move points. Using move points, players can make moves such as traversing the board, (accompany this point with an image of compass?) untampering hardware, making guesses/ accusations and revealing other players cards. During gameplay, a player may encounter a random event. Random events enable the user to gain information about the crime and learn about digital forensics. The first player to correctly make an accusation of the details of the crime wins the game.
